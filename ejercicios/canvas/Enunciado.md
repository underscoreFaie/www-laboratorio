# PROYECTO Canvas

El proyecto consiste en una aplicación web con temática libre con la condición de que el núcleo del desarrollo se apoye
en la interfaz Canvas API.

Se proponen las siguientes sugerencias (no vinculantes):

- Juegos interactivos
- Realidad virtual
- Visualización dinámica de conceptos matemáticos
- Visualización de audio
- Composiciones gráficas empleando curvas de Bezier

El desarrollo se realizará en HTML, CSS y JS nativos, sin bibliotecas ni frameworks auxiliares.

El contenido del proyecto constará de

- un fichero index.html con los elementos que conforman la estructura básicos de la aplicación. Opcionalmente, podrán incluirse
  documentos HTML adicionales.
- hojas de estilos donde se recogerán los estilos de la aplicación de forma que:
  - el diseño cromático y tipográfico esté independizando con claridad del resto de estilos
  - el diseño del espacio se realice aplicando de forma coherente técnicas de composición, posicionamiento y transformación
- módulos JS encargados de desarrollar la lógica de la aplicación separando con claridad las responsabilidades:
  - gestión de la interacción y la animación
  - definición de las clases representativas que participarán en los dibujos del lienzo

## Objetivos

El objetivo de esta actividad es el empleo de la interfaz Canvas API para el desarrollo de aplicaciones gráficas.

## Consideraciones

En la valoración del proyecto se tendrá en cuenta:

- la aplicación de las buenas prácticas explicadas a lo largo del curso
- la incorporación de elementos interactivos (panel con controles de ajustes, menús modales, etc.)
